# R1

This is to update the organisational-group table in the report so it compares Public Gen AI outcomes against the existing Copilot baseline, rather than showing Public Gen AI usefulness in isolation.

Could you please extract a group-level comparison of Copilot and Public Gen AI usage/usefulness from the survey data?

For each organisational group, please provide results split by Copilot access type where sample size allows:

- M365 Copilot
- Copilot Chat/basic
- Exclude or separately flag "unknown" Copilot access

Measures needed:

1. Copilot regular use
   - From Q13: share using Copilot weekly or more
   - Weekly or more = a few times a week + a few times a day + most of the day

2. Copilot usefulness
   - From Q15: share rating Copilot moderately useful or better
   - If possible, also provide very/extremely useful as a secondary cut

3. Public Gen AI regular use
   - From Q8: share using Public Gen AI tools weekly or more
   - Weekly or more = a few times a week + a few times a day + most of the day

4. Public Gen AI usefulness
   - Share rating at least one of ChatGPT, Gemini or Claude moderately useful or better
   - Use only respondents who used at least one public Gen AI tool and gave a valid usefulness response

5. Added value beyond Copilot
   - Share saying at least one Public Gen AI tool provided some/significant added value beyond Copilot, if available

Please include denominators for every cell, ideally as n used for the percentage, because some group x access-type cells may be small. If any cell has a very low n, please flag it rather than suppressing silently.

Suggested output format:

| Organisational group | Copilot access type | n | Copilot weekly+ | Copilot useful | Public AI weekly+ | Public AI useful | Added value beyond Copilot |
|---|---|---:|---:|---:|---:|---:|---:|
